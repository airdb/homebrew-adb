# Homebrew Taps
Airdb Homebrew Taps

## Installation

1. Install tools

```
brew tap aidb/taps
brew install adb

or

brew install airdb/taps/adb
```


```
```

```
brew edit airdb/taps/adb

```

or

```
vim /usr/local/Homebrew/Library/Taps/airdb/homebrew-taps/adb.rb
```
